====================================
		conv 
      General purpose converter
====================================


Conv converts input value to desired output value

Usage: conv [OPTION]... -i input_type -o desired_output_type [VALUE]

options:
	-h	--help
	-l	--printlist
	-p	--precision
	-v	--verbose


Supported conversions:
======================


	Number systems:
	---------------

	* Binary
	* Decimal
	* Octal
	* Hexadecimal 


	Systems of units of measurement:
	--------------------------------

	* Imperial system
	* SI system

	
	Temperature:
	------------

	* Celsius
	* Kelvin
	* Fahrenheit
