====================================
		conv 
      General purpose converter
====================================


Conv converts input value to desired output value

Usage: conv [OPTION]... -i input_type -o desired_output_type [VALUE]...


Supported conversions:
======================


	Number systems:
	---------------

	* Binary
	* Decimal
	* Octal
	* Hexadecimal 

	
	Temperature:
	------------

	* Celsius
	* Kelvin
	* Fahrenheit
	
	
	Measure:
	--------

	* Ounce to gram
